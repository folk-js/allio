## Installation

1. Install [Node + NPM](https://nodejs.org/en/download)
1. Install [Rust + Cargo](https://rust-lang.org/tools/install/)
1. Clone the [repo](https://github.com/folk-js/axio)
1. Run `npm i && npm run dev`

## MacOS

Once the dev server is running the Tauri icon will appear in the top toolbar that lets you select what type of desktop overlay you want.
